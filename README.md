# Easy Parking Project

## ขั้นตอนการติดตั้ง

1. ติดตั้ง Xampp บนเครื่องของคุณ
```
   net start MariaDB
```
2. Clone โปรเจคนี้:
   ```bash
   git clone -b Mockupdata https://github.com/your-repo/easy-parking.git
   cd Easy_Parking
   ```
## Firststep Use comands 
```
pip install -r requirements.txt
```
